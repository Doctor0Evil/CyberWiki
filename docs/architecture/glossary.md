# Architecture glossary

**Knowledge graph**: A structured collection of terms, documents, taxonomy mappings, and provenance records representing verified information about cybernetic and human–machine systems.[web:12][web:14]

**Taxonomy**: A hierarchical or faceted classification scheme that organizes entities into domains, device classes, protocols, and governance categories.

**Provenance**: Metadata describing sources, attestations, and verification events associated with each entity in the graph, including regulatory, industrial, and academic references.
