# ALN, KYC, and DID alignment

CyberWiki models identity, audit, and governance attributes so that its knowledge graph can integrate with ALN, KYC, and decentralized identity infrastructures. Entities can be linked to attestations, multi-signature approvals, and transaction logs that document how information was curated and verified.[web:35][web:38]

This design supports traceable compliance and auditability when CyberWiki is embedded into regulated workflows or citizen-facing assistance systems.
