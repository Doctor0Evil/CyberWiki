# Quantum cryptography practices

CyberWiki treats quantum-resilient cryptography as a first-class concern when modeling provenance and governance metadata. References to post-quantum signature schemes, key exchange mechanisms, and hybrid transitional deployments are documented as technical entities with explicit provenance and taxonomy labels.[web:55]

Where CyberWiki content refers to cryptographic controls that protect audit trails or communication channels, it documents the algorithms, parameter choices, and standardization status using verifiable references from recognized standards bodies or peer-reviewed sources.
