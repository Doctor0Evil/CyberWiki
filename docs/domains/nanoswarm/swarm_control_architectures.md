# Swarm control architectures

Swarm control architectures in CyberWiki describe how nanoscale or microscale agents are organized into controllable ensembles using layered coordination strategies.[web:71][web:74] Entries distinguish between fully centralized schemes, hierarchical control, and distributed approaches where agents rely on local rules and neighborhood information to achieve global objectives.[web:71][web:74]

For each architecture class, CyberWiki documents assumptions about communication bandwidth, timing, fault tolerance, and environmental variability, together with references to experimental and modeling studies. Security and robustness considerations, including resilience to compromised agents or adversarial interference with control channels, are treated as integral design dimensions.[web:71][web:74]
