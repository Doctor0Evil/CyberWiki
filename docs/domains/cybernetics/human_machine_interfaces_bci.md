# BCI-oriented human–machine interfaces

BCI-oriented human–machine interfaces within CyberWiki focus on how decoded neural or EEG-derived signals are mapped to interface elements, selection mechanisms, and feedback modalities.[web:66][web:73][web:76] Examples include speller interfaces, cursor control schemes, and assistive device controllers that translate user intent into discrete or continuous actions.[web:66][web:73][web:78]

Entries describe signal-to-command mappings, error correction strategies, latency and throughput characteristics, and user training requirements. Integration notes highlight how these interfaces attach to underlying control systems and safety mechanisms to prevent unintended actions.[web:66][web:73][web:76]
