# Feedback loops

Feedback loops in CyberWiki describe the mechanisms by which system outputs are measured and fed back into control logic to maintain stability, performance, and safety in cyber-physical and human–machine systems.[web:81][web:84] Documentation includes proportional–integral–derivative control, state feedback, adaptive control, and supervisory loops that coordinate multiple subsystems.[web:81][web:84]

Each entry specifies the measured variables, actuated quantities, update periods, and stability or robustness criteria used in design and verification. This allows practitioners to understand how feedback structures interact with sensors, actuators, and human operators in real deployments.[web:81][web:84]
