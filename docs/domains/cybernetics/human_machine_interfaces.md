# Human–machine interfaces

Human–machine interfaces in CyberWiki describe concrete mechanisms through which human operators observe system state and issue control actions in cyber-physical and information systems.[web:76][web:77][web:78][web:83] Entries cover display elements, input controls, alarm presentations, and interaction workflows in domains such as industrial control, medical devices, and assistive technologies.[web:76][web:80][web:83]

For each interface type, CyberWiki documents data types, update rates, error handling behaviors, and traceability hooks that support audit trails compliant with lifecycle and governance standards. Where applicable, references to standards such as ISA-101 and sector-specific guidance are included to align HMI design with safety and compliance expectations.[web:80][web:83]
