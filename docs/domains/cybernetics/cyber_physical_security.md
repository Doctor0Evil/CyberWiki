# Security for cyber-physical systems

Security documentation for cyber-physical systems in CyberWiki focuses on protecting the integrity, availability, and confidentiality of both digital and physical components.[web:86][web:90][web:93] Topics include threat models that span cyber and physical vectors, secure communication channels, access control to actuators, anomaly detection in sensor streams, and defense-in-depth strategies tailored to embedded and distributed environments.[web:86][web:90]

Entries summarize practices such as secure boot, hardware root of trust, cryptographic authentication of commands, and network segmentation, referencing publicly available guidance and research reports. Where formal verification or model-based assurance is used to support security claims, CyberWiki links to the underlying methods and assumptions.[web:86][web:90][web:93]
