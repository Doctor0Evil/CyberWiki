# Brain–computer interfaces

This section documents real-world brain–computer interface systems, including invasive and non-invasive device classes, signal acquisition hardware, stimulation technologies, and integration pathways to external computing systems. Each entry is grounded in published specifications, regulatory filings, or peer-reviewed studies.[web:12][web:14]
