# Integration of BCI with cybernetic systems

BCI–cybernetics integration in CyberWiki covers how decoded neural signals interface with control loops, human–machine interfaces, and cyber-physical systems.[web:88][web:90][web:93] Example patterns include using BCI outputs as supervisory commands in multi-loop control architectures, as input channels for assistive HMIs, or as triggers in safety-aware automation workflows.[web:88][web:90][web:93]

Documentation emphasizes signal scaling, filtering, and arbitration mechanisms that ensure BCI-derived commands coexist safely with conventional control inputs and automatic safety interlocks. Where appropriate, references to cyber-physical and safety governance documents provide context for acceptable use and verification practices.[web:86][web:90][web:93]
