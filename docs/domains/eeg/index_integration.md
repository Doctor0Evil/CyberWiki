# EEG domain integration overview

The EEG domain integration overview explains how acquisition pipelines, feature extraction methods, data standards, clinical standards, and open datasets interrelate inside CyberWiki. Entities are connected through shared taxonomy nodes and provenance records that capture their roles in diagnostic, research, and BCI-oriented workflows.

This structure allows users to navigate from a data standard to compatible datasets, from feature extraction methods to the pipelines that employ them, and from clinical standards to the device models and procedures that implement them. The result is a technically consistent picture of EEG practice grounded in verifiable references.
