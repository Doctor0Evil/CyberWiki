# EEG data standards and formats

EEG data standards in CyberWiki describe how electrophysiological recordings are stored, annotated, and shared in interoperable formats suitable for long-term analysis and regulatory review.[web:101][web:104] The documentation focuses on widely adopted standards such as BIDS for EEG and related conventions that specify file formats, metadata structures, and directory layouts.[web:101][web:104]

Entries summarize approved container formats, required and optional metadata fields, and typical usage patterns in open datasets and clinical repositories. Links to public electrophysiology datasets and neurophysiology standards help practitioners align acquisition and archiving practices with community norms.[web:101][web:104]
