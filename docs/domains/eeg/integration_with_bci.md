# Integration of EEG with BCI systems

This section describes how EEG acquisition pipelines, feature extraction methods, and data standards integrate into complete brain–computer interface systems.[web:66][web:68][web:100] CyberWiki documents architectures where EEG streams feed into decoding models, including classical pipelines and recent transformer- or TCN-based approaches for motor imagery and related tasks.[web:66][web:68][web:100]

Each integration profile specifies signal paths, preprocessing steps, model classes, and evaluation metrics, along with references to datasets and published results. The goal is to provide a clear, source-backed mapping from raw EEG data to BCI control outputs in real deployments.[web:66][web:68][web:100]
