# CyberWiki style guide

CyberWiki documentation is written in technically precise, non-fictional prose oriented toward cybernetics, BCI, EEG, nanoswarm, and associated governance systems. Sentences are direct, information-dense, and free of metaphor, speculation, or narrative framing.[web:20][web:30]

Every page must define its scope early, state assumptions explicitly, and use standard scientific terminology for math, geography, geology, and biophysics where applicable. All descriptions of devices, protocols, and architectures must correspond to real-world or formally specified systems.

Headings should be short and descriptive. Bullet lists are used sparingly in favor of clean paragraphs that read as if extracted from a technically edited research report or engineering dossier.
