# Guides

The guides section collects authoring, taxonomy, style, and API usage guidance for CyberWiki contributors and integrators. These documents describe how to structure content, align terminology with established standards, and integrate external systems without weakening the project’s non-fictional guarantees.[web:20][web:30][web:64]

Readers should start with the authoring standards and style guide, then consult taxonomy, API, and governance documents as needed for specific contribution or integration workflows.
