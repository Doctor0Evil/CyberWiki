# Taxonomy and tagging guide

CyberWiki uses a layered taxonomy to classify entities by domain, device class, protocol family, and governance category. Each term is assigned at least one taxon, ensuring that queries can be scoped to coherent technical subsets such as invasive BCI devices, EEG acquisition chains, or nanoswarm control protocols.[web:50][web:51]

Tags complement taxonomy nodes by providing flexible, case-normalized labels that support cross-cutting queries, such as all entities related to real-time control, safety certification, or quantum-resistant cryptography. Contributors should align new taxonomy entries and tags with existing structures whenever possible to maintain a stable, predictable navigation model.
