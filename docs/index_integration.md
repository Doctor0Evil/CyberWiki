# Cross-domain integration overview

The cross-domain integration overview describes how CyberWiki links BCI, EEG, cybernetics, nanoswarm, and governance documentation into a coherent knowledge graph.[web:88][web:90][web:93] Entities are connected via taxonomy membership, shared provenance records, and explicit relationship edges that reflect real technical couplings, such as an EEG pipeline feeding a BCI decoder embedded in a cyber-physical control system.[web:88][web:90][web:93]

This overview explains how integrators and AI-Chat clients can traverse the graph from devices to data standards, safety profiles, and governance artifacts. The aim is to support end-to-end reasoning about complex, human-integrated technologies using only verifiable, non-fictional information.[web:88][web:90][web:93]
