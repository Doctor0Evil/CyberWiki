# CyberWiki documentation

CyberWiki is a Rust-based documentation and assistance system constrained to non-fictional knowledge about cybernetics, brain–computer interfaces, EEG systems, nanoswarm architectures, and associated governance frameworks.[web:37]

The documentation set is organized as a technical reference for engineers, researchers, and regulators who require precise descriptions of devices, protocols, data flows, and safety constraints. Every page is written in an objective, professionally edited tone.

This index routes to domain-specific sections, architectural notes, API semantics, and authoring standards that govern how new content is introduced and validated inside the CyberWiki knowledge graph.
