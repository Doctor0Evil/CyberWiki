# Contributing to CyberWiki

CyberWiki welcomes contributions that improve the precision, coverage, and trustworthiness of its non-fictional documentation about cybernetics, BCI, EEG, nanoswarm systems, and governance frameworks.[web:31][web:35][web:38]

Contributions must adhere to the project scope and exclude speculative, fictional, or improvised backyard experimental systems. All submissions should reference verifiable sources such as regulatory documents, peer-reviewed publications, or manufacturer specifications.

Before opening a pull request, run `cargo fmt`, `cargo clippy`, and `cargo test` across the workspace. Proposed changes to documentation should follow the style guide under `docs/guides/style_guide.md` and maintain clear, technically edited prose.

By participating in this project, you agree to abide by the repository license and any applicable codes of conduct linked from the project homepage.
