# CyberWiki

CyberWiki is a Rust-based documentation and assistance system dedicated to rigorous, non-fictional coverage of cybernetics, brain–computer interfaces, EEG systems, nanoswarm architectures, and related governance frameworks.[web:37][web:49]

The project is structured as a Rust workspace with a core domain crate, a CLI crate, and an API crate, allowing deployment as a headless knowledge engine, a terminal tool, or a networked documentation service. All layers enforce a professional, technically edited style designed for engineers, researchers, and regulators.

AI-Chat integrations connect to the CyberWiki API and operate strictly within the constraints of the knowledge graph, taxonomy, and provenance model, ensuring that responses remain grounded in verifiable sources and avoid speculative or fictional content.
